## PARAMETERS ##


--packetsize <size>
Set the size of the packet to send [default: 65432];

--threads <threads>
The number of UDP client threads to create [default: 10];

--ip <ip>
The IP address to which the packets will be sent to [default: 142.250.184.163 (AKA google.pt)];

--port <port>
The port to which the packets will be sent to [default: 100];

--nowait
Doesn't show the config and skips the wait time [default: wait];

--nooutput
Hides the "Sent packet..." messages, which might make the program run faster [default: show output];
